# astroship
Зерттеу Күн жүйесіндегі астероидтардың қозғалысын зерттеп, бақылап қана қоймай, белгілі дәрежеде басқаруға болатынын көрсетуге бағытталған. Мысалы, лазер арқылы аздап әсер етіп, астероидтың траекториясын өзгертуге болады. Жобаның идеясы – астероидтарды жай ғарыш объекті емес, табиғи қозғалысқа ие және басқаруға болатын платформа ретінде қарастыру.
